# familydoctor-talk

Isolated staging application for TheFamilyDoctor.AI Talk.

This repository is intentionally separate from production frontend/backend repositories and infrastructure. Stages 1 and 2 build only a placeholder conversation shell: no real AI, speech-to-text, MuseTalk, backend writes, payments, auth, prescription workflow, or production data access.

## Local commands

```bash
npm install
npm run dev
npm run build
```
